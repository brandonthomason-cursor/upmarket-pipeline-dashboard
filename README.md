# 2026 Upmarket Pipeline and Revenue Dashboard

CFO-focused dashboard for partner-sourced upmarket deals, aligned with Ryan Roccon's WBR metric preferences.

## Features

- ✅ Pipeline progression metrics (qualified pipeline, conversion rates)
- ✅ Pacing metrics (pipeline pacing vs. targets, activity pacing)
- ✅ Rotating "Most Important Metrics" section
- ✅ Partner performance tracking
- ✅ Book of Business overlap detection
- ✅ Auto-updates every 6 hours via GitHub Actions
- ✅ Coda embed ready
